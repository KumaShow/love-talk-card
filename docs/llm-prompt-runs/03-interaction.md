# Interaction｜互動理解 Prompt

你現在是繁體中文戀愛對話卡牌編輯。請為主題「互動理解 / interaction」生成卡片文案。

這個主題的定義：
- 核心情緒與對話目的：
  - 核心情緒：共感、調整、協作。
  - 對話目的：觀察「我們如何相處」，優化溝通模式與日常生活品質。
  - 主題重點：重點是「我們怎麼互動」，不是單純談個人本質。
- 適合問的內容類型：
  - 生活習慣。
  - 吵架後的和好偏好。
  - 理想的約會方式。
  - 金錢觀分配。
  - 家事分工。
  - 愛的訊息接收方式。
- 不適合問的內容類型：
  - 避免涉及個人深層心理或創傷性話題。
  - 避免變成「批鬥大會」或「翻舊帳」，不討論「誰對誰錯」，只討論「我們怎麼做會更好」。
  - 原則底線與價值觀審判。
  - 忠誠與背叛。
- intimate 與 base 的差別：
  - Base 題（基礎題）：
    - 屬性：溫馨、有趣、適合在任何地方（咖啡廳、散步中）進行。
    - 對話目標：增加理解與共感，幫助雙方知道彼此怎麼相處更舒服。
  - Intimate 題（私密題）：
    - 屬性：高感性、高脆弱性、適合安靜的私人空間（睡前、居家）。
    - 差別原則：
      - 深度不同：Intimate 題會更靠近不安、誤解、修復與被接住的感受。
      - 互動焦點：可以談親密情境中的表達、回應、安撫與接住方式，但不要露骨。
      - 優先權：幫雙方知道「怎麼互動會更貼近」。

請生成共 20 張卡：
- 15 張 base
- 5 張 intimate

配比要求：
- base：level 1 = 5 張、level 2 = 6 張、level 3 = 4 張
- intimate：level 2 = 2 張、level 3 = 3 張

文案要求：
- 只產出繁體中文 `zh-TW`
- 每題一句為主，必要時可兩小句，但不要冗長
- 每題只問一件事，避免雙重提問
- 要自然口語、適合伴侶直接念出來
- 避免過度抽象，例如「你如何定義愛」這種太大題
- intimate 卡請依照上方「差別原則」撰寫，保持私密但不要露骨
- 題型要分散：表達方式、傾聽感受、誤解、修復、互動節奏
- 避免同一個切角重複太多次
- 避免和其他主題混題：
  - **Interaction vs. Self**
    - 混淆點：都涉及行為。
    - 區分：Self 是「我為什麼是這樣的人（個人本質）」；Interaction 是「當我遇到你，我們產生了什麼反應（關係動態）」。
    - ❌「你是個沒有安全感的人嗎？」
      這是 self。
    - ✅「當你沒有安全感時，希望我怎麼做？」
      這是 interaction。
- Level 遞進：
  - Level 1｜日常相處偏好
    - 怎樣陪伴最舒服。
    - 最喜歡的約會節奏。
    - 吵架後希望多久和好。
    - 從低壓力的互動習慣與陪伴方式切入。
  - Level 2｜溝通與磨合
    - 哪種說話方式最讓你受傷。
    - 哪件小事最容易讓你不開心。
    - 你希望我怎麼哄你。
    - 開始處理互動中的情緒訊號與理解差異。
  - Level 3｜更真實的相處修復
    - 我最常忽略你的需求是什麼。
    - 如果我們總是為同件事吵架，你覺得原因是什麼。
    - 你最希望我改變的一件小事。
    - 更靠近修復、調整與被理解的需求，但不要變成批判對方。

  - 範例遞進表：

    | 主題 | Level 1 | Level 2 | Level 3 |
    | :--- | :--- | :--- | :--- |
    | **Interaction** | 我們兩人之中，誰比較愛賴床？ | 當我們意見不合時，你最希望我第一時間怎麼做？ | 你覺得我們在相處上，最需要一起克服的挑戰是什麼？ |

id_prefix | level | isIntimate | 文案 | 設計理由

- isIntimate 請填 true / false
