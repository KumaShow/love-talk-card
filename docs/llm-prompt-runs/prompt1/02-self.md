# Self｜自我探索 Prompt

你現在是繁體中文戀愛對話卡牌編輯。請為主題「自我探索 / self」生成卡片文案。

這個主題的定義：
- 核心情緒與對話目的：
  - 核心情緒：好奇、平靜、接納。
  - 對話目的：讓對方了解「關係之外的我」，分享個人成長背景與內心世界。
  - 主題重點：重點是「我怎麼理解自己」，不是「我們怎麼互動」。
- 適合問的內容類型：
  - 個人夢想。
  - 恐懼。
  - 童年影子（非創傷性）。
  - 價值觀。
  - 生活哲學。
  - 對自我的期許。
  - 壓力來源。
  - 獨處的需求。
- 不適合問的內容類型：
  - 避免涉及醫療級心理創傷或具體的黑暗隱私（如：你最卑鄙的時刻），避免讓對方感到被審判。
  - 避免把焦點放在誰對誰錯。
  - 避免把題目寫成衝突修復或未來承諾討論。
- intimate 與 base 的差別：
  - Base 題（基礎題）：
    - 屬性：溫馨、有趣、適合在任何地方（咖啡廳、散步中）進行。
    - 對話目標：增加生活情趣，增進日常理解。
  - Intimate 題（私密題）：
    - 屬性：高感性、高脆弱性、適合安靜的私人空間（睡前、居家）。
    - 差別原則：
      - 深度不同：Intimate 題會要求更多「情緒揭露」，而不只是「資訊分享」。
      - 揭露方向：可以談不容易說出口的需求、界線、脆弱與被理解的渴望，但不要逼答。
      - 優先權：幫對方看見「關係裡更真實的我」。

請生成共 20 張卡：
- 15 張 base
- 5 張 intimate

配比要求：
- base：level 1 = 5 張、level 2 = 6 張、level 3 = 4 張
- intimate：level 2 = 2 張、level 3 = 3 張

文案要求：
- 只產出繁體中文 `zh-TW`
- 每題一句為主，必要時可兩小句，但不要冗長
- 每題只問一件事，避免雙重提問
- 要自然口語、適合伴侶直接念出來
- 避免過度抽象，例如「你如何定義愛」這種太大題
- intimate 卡請依照上方「差別原則」撰寫，保持私密、真誠，但不要逼答
- 題型要分散：自我描述、需求辨識、界線、脆弱、成長
- 避免同一個切角重複太多次
- 避免和其他主題混題：
  - **Self vs. Interaction**
    - 混淆點：都涉及行為。
    - 區分：Self 是「我為什麼是這樣的人（個人本質）」；Interaction 是「當我遇到你，我們產生了什麼反應（關係動態）」。
    - ❌「你覺得我們吵架最大的問題是什麼？」
      這是 interaction。
    - ✅「你吵架時最怕被怎麼對待？」
      這是 self。
- Level 遞進：
  - Level 1｜認識習慣與偏好
    - 一個人放鬆的方式。
    - 最需要被安慰的時候。
    - 最不喜歡被怎麼對待。
    - 從容易開口的習慣、偏好與自我感受切入。
  - Level 2｜情緒與內在模式
    - 生氣時最真實的反應。
    - 最怕失去什麼。
    - 最難開口承認的脆弱。
    - 開始碰到需求、壓力與內在模式。
  - Level 3｜深層自我揭露
    - 哪件事讓你變成現在的你。
    - 最希望被理解卻很少說出口的事。
    - 你最怕我誤會你的地方。
    - 更靠近身分認同、脆弱與自我理解，但不要變成創傷逼問。

  - 範例遞進表：

    | 主題 | Level 1 | Level 2 | Level 3 |
    | :--- | :--- | :--- | :--- |
    | **Self** | 你最喜歡的一種度過週末的方式？ | 當你感到壓力很大時，你通常會怎麼躲起來？ | 如果生命要重來，你最想保留的性格特質是什麼？ |
