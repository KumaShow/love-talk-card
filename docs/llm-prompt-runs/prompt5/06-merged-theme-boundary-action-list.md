# 主題邊界修正整合清單

整合來源：

- `docs/llm-prompt-runs/05-theme-boundary-review.md`
- `docs/llm-prompt-runs/05-cross-theme-boundary-review.md`

## 整合原則

1. 兩份審查都點名、且主題漂移明顯的卡片，列為必修。
2. 只有一份點名、但邊界語彙需要收斂的卡片，列為 P1 微調。
3. 偏向題型去重或主題純度提升的，留到第二輪處理。

## 一、必修清單

| 優先 | 卡片 | 問題 | 修正方向 |
|---|---|---|---|
| P0 | `att-009` | `attraction` 滑向 `trust` | 從「在意程度」拉回「魅力變強的瞬間」 |
| P0 | `sel-019` | `self` 滑向 `interaction` | 從「不太會開口」拉回「我怎麼藏需求」 |
| P0 | `int-016` | `interaction` 滑向 `trust` | 從「被接住」改成「怎樣比較願意說下去」 |
| P0 | `int-020` | `interaction` 滑向 `trust` | 從「被放在心上」改成「被聽進去、被理解」 |
| P0 | `tru-011` | `trust` 滑向 `interaction` | 從「修復流程」改成「什麼能守住信任」 |
| P0 | `tru-016` | `trust` 滑向 `self` | 從「脆弱內容」拉回「怎樣被可靠接住」 |

## 二、P1 邊界微調

這批保留在原主題，但需要微調語彙，避免讀者誤判主題。

| 卡片 | 判斷 | 修正方向 |
|---|---|---|
| `int-009` | 保留在 `interaction`，但「真的被懂」容易帶出安全感語彙 | 保留「我做什麼反應」的互動核心，改得更像傾聽反應 |
| `sel-007` | 保留在 `self`，但「怎麼反應」容易被讀成外顯互動 | 從「反應」改成「心裡先冒出的感覺」 |
| `int-018` | 保留在 `interaction`，但容易滑向修復後的內在狀態 | 聚焦和好後的互動節奏 |
| `tru-017` | 保留在 `trust`，但可補強依賴與安心 | 讓「放心依賴」更直接 |
| `tru-019` | 保留在 `trust`，但可補強不安時的安全感 | 讓「怎麼靠近才安心」更明確 |

## 三、第二輪主題純度候選

這批不是明顯錯題，也不包含上方 P1。後續若要再精修，修完主題會更乾淨。

| 主題 | 卡片 | 建議 |
|---|---|---|
| `attraction` | `att-011` `att-016` `att-019` `att-020` | 減少「互動方式」感，強化「心動／曖昧／吸引」 |
| `self` | `sel-016` `sel-017` `sel-020` | 減少對方視角，拉回自我理解與自我照顧 |
| `interaction` | `int-006` `int-010` `int-015` | 把題目更明確落在互動節奏、回應方式、分歧處理 |
| `trust` | `tru-007` | 把「安心、依賴、被接住」說得更直接 |

## 四、去重整理清單

這些不一定是跑題，但很容易撞角度。

| 優先 | 題組 | 重複核心 | 建議 |
|---|---|---|---|
| P1 | `int-010` / `tru-011` | 誤會發生後，希望對方先怎麼做 | 保留 `int-010`，重寫 `tru-011` |
| P1 | `int-006` / `int-016` | 難過時，希望對方怎麼回應 | `int-016` 改成互動入口 |
| P1 | `sel-016` / `sel-019` | 需要被照顧、被溫柔對待 | `sel-019` 聚焦隱藏需求 |
| P2 | `att-016` / `att-019` | 靠近情境與氣氛 | 一張留情境，一張留動作 |
| P2 | `att-001` / `att-005` | 初始吸引與注意到的細節 | 留一張較直覺的，另一張改更具體 |
| P2 | `int-009` / `int-010` / `int-015` | 溝通卡住時怎麼處理 | 分成傾聽、理解分歧、卡住後回來 |
| P2 | `tru-016` / `tru-017` / `tru-019` | 脆弱、依賴、不安時的支撐 | 明確拆成三個不同角度 |

## 五、建議修正順序

1. 先修 `tru-011`、`int-016`、`int-020`
2. 再修 `sel-019`、`att-009`、`tru-016`
3. 接著微調 `int-009`、`sel-007`
4. 第二輪再做其餘主題純度提升
5. 最後才做去重整理
