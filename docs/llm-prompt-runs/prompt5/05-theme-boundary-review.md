# 主題邊界修訂建議稿（Attraction / Self / Interaction / Trust）

## 文件用途
這份文件整理四個主題的邊界校正結果，目的是提供內容編輯與後續出題時的具體修訂依據。
原則是**先修邊界、再修題意**，避免卡片落在錯誤主題、造成題庫重疊或難以辨識主題核心。

---

## 一、四個主題的邊界定義摘要

### 1. Attraction
**核心**：被吸引、心動、想靠近。
重點落在第一印象、外在細節、可愛小動作、曖昧張力。
**最常混到**：interaction，其次 trust。

### 2. Self
**核心**：我怎麼感受自己、理解自己。
重點落在壓力、自在、不安、自我認同、成長願望、內在狀態。
**最常混到**：interaction，其次 trust。

### 3. Interaction
**核心**：兩個人怎麼相處比較順。
重點落在聊天方式、做決定、衝突處理、節奏、距離、配合方式。
**最常混到**：trust。

### 4. Trust
**核心**：能不能放心把自己交給對方。
重點落在安心感、可靠性、守住承諾、脆弱時被接住、出錯後能否修復。
**最常混到**：interaction。

## 二、各主題優先修正卡片與原因

## Attraction
### 優先修正
- **att-009**
- **att-011**
- **att-016**
- **att-019**
- **att-020**

### 修正原因
這批卡片的問題主要是把「心動與靠近」寫得太像相處互動，容易滑到 interaction；部分題目又帶到對方反應與陪伴方式，會讓 trust 感過重。
優先修正這些卡片，能先把 Attraction 的主軸拉回「吸引感」本身。

### 建議改寫
- **att-009**：有沒有哪個瞬間，讓你突然發現自己比想像中還想靠近我？
- **att-011**：當我靠近你時，哪一個小動作最容易讓你心動？
- **att-016**：只有我們兩個人的時候，哪一種靠近最容易讓你心動？
- **att-019**：什麼樣的氣氛，最容易讓你想再靠近我一點？
- **att-020**：有沒有某個時刻，你其實很希望我主動靠近你一下？

---

## Self
### 優先修正
- **sel-019**
- **sel-020**
- 次修：**sel-016**
- 次修：**sel-017**

### 修正原因
Self 的題目需要更聚焦在「我怎麼看待自己、照顧自己、理解自己」。
這批卡片容易被讀成希望對方怎麼做，因而偏向 interaction 或 trust；先修高風險題，能把 Self 的內在視角重新固定住。

### 建議改寫
- **sel-019**：你有沒有哪種時候，明明很需要被照顧，卻還是會先自己撐著？
- **sel-020**：如果現在可以放心對自己說一句真心話，你最想承認什麼？
- **sel-016**：你通常在什麼時候，最需要多給自己一點溫柔？
- **sel-017**：有沒有一個只有自己比較知道的樣子？

---

## Interaction
### 優先修正
- **int-006**
- **int-009**
- **int-010**
- **int-015**
- **int-016**
- **int-018**

### 修正原因
Interaction 的核心是「相處方式」而不是「安不安心」；這批題目需要更明確地落在溝通節奏、互動回應、協調方式、和好後的處理流程。
先修這些卡片，可以避免題目表面像互動，實際卻跑去 trust。

### 建議改寫
- **int-006**：當我們其中一個心情不好時，你比較希望對方先開口，還是先留一點空間？
- **int-009**：我回你訊息時，哪一種回法最能讓對話繼續下去？
- **int-010**：如果我們對同一件事理解不同，你希望我先問清楚，還是先把我的想法說完？
- **int-015**：我們意見卡住之後，你比較希望先停一下，還是馬上把問題拆開談？
- **int-016**：當我們其中一個難過時，你比較希望對方怎麼拿捏靠近的距離？
- **int-018**：和好之後，你會想先恢復聊天，還是先各自安靜一下？

---

## Trust
### 優先修正
- **tru-007**
- **tru-011**
- **tru-016**
- **tru-017**
- **tru-019**

### 修正原因
Trust 的題目必須穩穩落在「能否放心依賴、能否被接住、能否修復誤會」。
這批卡片有些還不夠直接，容易被看成一般相處提問；修正後可以更明確把「信任」說清楚。

### 建議改寫
- **tru-007**：你遇到挫折時，什麼樣的陪伴最能讓你安心開口？
- **tru-011**：如果我們之間出現誤會，你最希望我先怎麼做，才會讓你比較放心？
- **tru-016**：當你有比較脆弱的地方時，你最希望我怎麼對待，才會覺得安心？
- **tru-017**：你有沒有哪個時刻，其實很想放心依賴我？
- **tru-019**：當你開始不安時，什麼樣的陪伴最能讓你安心？

## 三、重複題意整理與刪修建議

- **sel-019 與 interaction-017**
- **sel-016 與 trust-016 / interaction-016**

## 四、後續編輯原則
1. **先判定主題，再寫題目**：每張卡先確認只服務一個核心主題。
2. **避免雙主題敘述**：若題目同時問「我怎麼想」與「你怎麼做」，通常就會混題。
3. **優先保留最直接的卡**：同義或近義卡只留一張最清楚的，其餘做切分或刪修。
4. **把抽象詞具體化**：像「安心」「靠近」「溫柔」這類詞，必須落到可辨識的情境或行為。
5. **最小修正，不做全面重寫**：先修邊界與重複題意，保留原卡風格與語氣，再視需要微調。

---

## 結語
這份文件主張的是**最小修正**，不是全面重寫。
後續處理原則應以「先修邊界、再修重複、最後才做語氣微調」為準，避免把原本可用的卡片整批改掉。
