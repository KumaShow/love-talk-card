# attraction 修改方向建議

**版本**：2026-06-04  
**範圍**：`src/data/themes/attraction.json`  
**目的**：整理 `attraction` 主題目前較接近、容易重複的區塊，提供後續改稿時的優先順序與方向。

---

## 整體判斷

`attraction` 的核心方向是對的，題目仍然圍繞「心動、吸引、被看見、曖昧氣氛」。  
但目前的問題不是跑題，而是**同一種心動角度排得太密**，尤其是第一眼、瞬間心動、重新吸引、靠近感這幾條線，容易讓玩家覺得彼此只是換句話說。

建議的修改原則是：

1. 保留 `attraction` 的浪漫與曖昧感。
2. 把「第一印象」「日常欣賞」「動作吸引」「心動瞬間」「私密靠近」拆開。
3. 每張卡只保留一個明確角度，避免同題重複。

## 建議的分流方式

| 區段 | 建議保留的方向 | 建議避免的方向 |
|---|---|---|
| Level 1 | 第一印象、日常欣賞、初始期待 | 太早進入「重新認識一次」或「想靠近」 |
| Level 2 | 最近的心動、具體行為吸引、說話風格 | 多張都問同一種「你覺得我很迷人嗎」 |
| Level 3 | 曖昧升溫、重新被吸引、少說出口的喜歡 | 連續堆疊「靠近」「想靠近」「只剩我們兩個」 |
| `isIntimate: true` | 柔和、邀請式、可保留溫度 | 直接碰觸、強烈身體召喚、太明確的親密暗示 |

---

## 修改優先順序

1. 先拆 `att-001-base` / `att-005-base` 的重複視角。
2. 再拆 `att-006-base` / `att-009-base` / `att-010-base` 的心動密度。
3. 接著收斂 `att-012-base`，避免和前段卡重複。
4. 再處理 `att-016-intimate` / `att-019-intimate`，讓私密段落不要只剩「靠近」。
5. 最後才評估 `att-017-intimate` / `att-018-intimate` / `att-020-intimate` 是否要延後解鎖。

---

## 結論

`attraction` 不需要整體重寫，真正需要的是**把同類型的心動題拆散**。  
只要把第一眼、日常欣賞、瞬間心動、曖昧靠近、私密暗示這五條線分開，整個主題就會更有層次，也更不容易讓玩家覺得重複。
