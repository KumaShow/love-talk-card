# Attraction｜心動瞬間 Prompt

你現在是繁體中文戀愛對話卡牌編輯。請為主題「心動瞬間 / attraction」生成卡片文案。

這個主題的定義：
- 核心情緒與對話目的：
  - 核心情緒：曖昧感、心動感、喜歡感、被吸引的感覺。
  - 對話目的：重新感受「為什麼會喜歡你」、「我為何被你吸引」，強化正向反饋與戀愛感。
  - 主題重點：讓伴侶回到關係中的吸引力，而不是討論關係經營。這個主題的重點是喜歡，不是承諾。
- 適合問的內容類型：
  - 初見面的印象。
  - 對方閃閃發光的時刻。
  - 特定的外在或內在特質。
  - 被對方在乎的小細節。
  - 身體的化學反應。
  - 最喜歡對方的習慣、表情或反應。
  - 對方讓自己覺得特別的瞬間。
- 不適合問的內容類型：
  - 嚴禁問「你覺得我哪裡需要改進」或「前任哪裡比我好」，這會破壞心動感。
  - 關係中的安全感問題。
  - 承諾、忠誠、信任議題。
  - 吵架、失望、受傷經驗。
  - 價值觀衝突。
  - 原生家庭創傷。
  - 控制感、佔有欲。
- intimate 與 base 的差別：
  - Base 題（基礎題）：
    - 屬性：溫馨、有趣、適合在任何地方（咖啡廳、散步中）進行。
    - 對話目標：增加生活情趣，增進日常理解。
  - Intimate 題（私密題）：
    - 屬性：高感性、高脆弱性、適合安靜的私人空間（睡前、居家）。
    - 差別原則：
      - 深度不同：Intimate 題會要求更多「情緒揭露」，而不只是「資訊分享」。
      - 親密感來源：可以涉及想靠近、觸碰、渴望與身體記憶，但要保持優雅，不要露骨。
      - 優先權：更強調「只有你會讓我這樣心動」的獨特性。

請生成共 20 張卡：
- 15 張 base
- 5 張 intimate

配比要求：
- base：level 1 = 5 張、level 2 = 6 張、level 3 = 4 張
- intimate：level 2 = 2 張、level 3 = 3 張

文案要求：
- 只產出繁體中文 `zh-TW`
- 每題一句為主，必要時可兩小句，但不要冗長
- 每題只問一件事，避免雙重提問
- 要自然口語、適合伴侶直接念出來
- 避免過度抽象，例如「你如何定義愛」這種太大題
- intimate 卡請依照上方「差別原則」撰寫，保持私密但不要露骨
- 題型要分散：回憶型、感受型、觀察型、現在式、未來期待型
- 避免同一個切角重複太多次
- 避免和其他主題混題：
  - **Attraction vs. Trust**
    - 混淆點：都涉及對另一半的感受。
    - 區分：Attraction 是「我看著你時眼裡的火花」；Trust 是「我背對著你時心裡的安穩」。
    - ❌「你覺得我是能陪你走很久的人嗎？」
      這是 trust。
    - ✅「最近一次你突然很想靠近我是什麼時候？」
      這是 attraction。
- Level 遞進：
  - Level 1｜輕鬆回憶
    - 第一次印象。
    - 最可愛的地方。
    - 哪次最心動。
    - 描述客觀事實、過去的記憶或明顯的偏好。
    - → 好回答、低壓力、容易開場。
  - Level 2｜更具體的情感辨識
    - 哪種樣子最吸引你。
    - 哪件小事讓你印象很深。
    - 有沒有偷偷吃醋過。
    - 描述當下的心情、行為背後的動機。
    - → 開始進入真實情緒。
  - Level 3｜私密但不沉重
    - 如果重新遇見，還會喜歡我嗎。
    - 哪個瞬間讓你差點更愛上我。
    - 你最不敢承認自己很在意我的地方是什麼。
    - 更貼近深層吸引、被渴望與持續心動的感受，但不要變成承諾討論。
    - → 更貼近 intimate 區域。

  - 範例遞進表：

    | 主題 | Level 1 | Level 2 | Level 3 |
    | :--- | :--- | :--- | :--- |
    | **Attraction** | 第一次見面時對我印象最深的地方？ | 最近一次覺得我「很有魅力」的瞬間？ | 你最不敢承認自己其實很想靠近我的時刻是什麼？ |
