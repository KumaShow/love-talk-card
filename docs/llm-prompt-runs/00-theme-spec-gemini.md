# Gemini 主題規格說明
---

### A. 每個主題最核心的情緒與對話目的

| 主題 | 核心情緒 | 對話目的 |
| :--- | :--- | :--- |
| **1. attraction** | **欣賞、傾心、喜悅** | 重新確認「我為何被你吸引」，強化正向反饋與戀愛感。 |
| **2. self** | **好奇、平靜、接納** | 讓對方了解「關係之外的我」，分享個人成長背景與內心世界。 |
| **3. interaction** | **共感、調整、協作** | 觀察「我們如何相處」，優化溝通模式與日常生活品質。 |
| **4. trust** | **安穩、承諾、歸屬** | 處理「脆弱與不確定性」，建立深度安全感與長期共識。 |

---

### B. 每個主題適合問的內容類型

1.  **attraction｜心動瞬間**
    *   初見面的印象、對方閃閃發光的時刻、特定的外在或內在特質、被對方在乎的小細節、身體的化學反應。
2.  **self｜自我探索**
    *   個人夢想、恐懼、童年影子（非創傷性）、價值觀、生活哲學、對自我的期許、壓力來源、獨處的需求。
3.  **interaction｜互動理解**
    *   生活習慣、吵架後的和好偏好、理想的約會方式、金錢觀分配、家事分工、愛的訊息接收方式。
4.  **trust｜信任成長**
    *   對承諾的定義、對未來的共同想像、在對方面前展現脆弱的感受、如何處理嫉妒或不安、支持系統的建構。

---

### C. 每個主題不適合問的內容類型

1.  **attraction**：嚴禁問「你覺得我哪裡需要改進」或「前任哪裡比我好」，這會破壞心動感。
2.  **self**：避免涉及醫療級心理創傷或具體的黑暗隱私（如：你最卑鄙的時刻），避免讓對方感到被審判。
3.  **interaction**：避免變成「批鬥大會」或「翻舊帳」，不討論「誰對誰錯」，只討論「我們怎麼做會更好」。
4.  **trust**：嚴禁「忠誠度測試」（如：如果你媽跟我掉進水裡）或帶有威脅感的假設。

---

### D. 四個主題彼此最容易混淆的地方

*   **Attraction vs. Trust**：
    *   *混淆點*：都涉及對另一半的感受。
    *   *區分*：Attraction 是「我看著你時眼裡的火花」；Trust 是「我背對著你時心裡的安穩」。
*   **Self vs. Interaction**：
    *   *混淆點*：都涉及行為。
    *   *區分*：Self 是「我為什麼是這樣的人（個人本質）」；Interaction 是「當我遇到你，我們產生了什麼反應（關係動態）」。

---

### E. 每個主題的 level 1 / level 2 / level 3 遞進邏輯

我們將難度（親密度）分為三個階段：

*   **Level 1（破冰與觀察）**：描述客觀事實、過去的記憶或明顯的偏好。
*   **Level 2（感受與理解）**：描述當下的心情、行為背後的動機。
*   **Level 3（靈魂與未來）**：探討深層價值觀、恐懼、對未來的共同想像。

**範例遞進表：**

| 主題 | Level 1 | Level 2 | Level 3 |
| :--- | :--- | :--- | :--- |
| **Attraction** | 第一次見面時對我印象最深的地方？ | 最近一次覺得我「很有魅力」的瞬間？ | 你覺得我們之間最無法被取代的連結是什麼？ |
| **Self** | 你最喜歡的一種度過週末的方式？ | 當你感到壓力很大時，你通常會怎麼躲起來？ | 如果生命要重來，你最想保留的性格特質是什麼？ |
| **Interaction** | 我們兩人之中，誰比較愛賴床？ | 當我們意見不合時，你最希望我第一時間怎麼做？ | 你覺得我們在相處上，最需要一起克服的挑戰是什麼？ |
| **Trust** | 你覺得我是個守信用的人嗎？ | 在什麼情況下，你會覺得「有你在真好」？ | 當你感到最脆弱時，你相信我會如何接住你嗎？ |

---

### F. Intimate 題與 Base 題的差別原則

在每個主題中，我們都可以穿插 Intimate（私密題），其設計原則如下：

1.  **Base 題（基礎題）**：
    *   **屬性**：溫馨、有趣、適合在任何地方（咖啡廳、散步中）進行。
    *   **對話目標**：增加生活情趣，增進日常理解。
2.  **Intimate 題（私密題）**：
    *   **屬性**：高感性、高脆弱性、適合安靜的私人空間（睡前、居家）。
    *   **差別原則**：
        *   **深度不同**：Intimate 題會要求更多「情緒揭露」，而不只是「資訊分享」。
        *   **身體連結**：Intimate 題可以涉及身體觸覺與親密感受（如：你最喜歡我怎麼抱你？），但保持優雅。
        *   **優先權**：Intimate 題會觸及「非你不可」的獨特性。

---

**編輯手記：**
為了確保「不拗口」且「像人在說話」，建議在設計文案時，多使用**「你覺得...」、「我好奇...」、「有沒有過一個時刻...」**這樣的開頭，避免使用「請列舉」、「請分析」等測驗式字眼。
