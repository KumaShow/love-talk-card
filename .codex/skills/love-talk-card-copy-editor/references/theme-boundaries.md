# 主題邊界

決定卡牌是否符合戀愛卡牌分類法時參考本文件。

## attraction（吸引力）

- 核心：火花、慾望、吸引力、被看見、浪漫、持續的心動時刻。
- 好角度：第一印象、磁性特質、依然吸引的習慣、難忘的化學作用、溫柔的期待。
- 避免：未來承諾規劃、衝突修復、深層自我分析、關係治理。

## self（自我）

- 核心：在關係中的自我認知、需求、邊界、脆弱、內在狀態、個人成長。
- 好角度：我需要什麼、我隱藏什麼、什麼幫助我感到像自己、我正在學習關於自己的什麼。
- 避免：團隊工作流程、衝突機制、純粹伴侶未來談話、求愛的火花。

## interaction（互動）

- 核心：我們如何溝通、回應、傾聽、彼此錯過、修復及相互理解。
- 好角度：感到被聽見、道歉、時機、表達風格、誤讀信號、修復嘗試。
- 避免：主要是內部自我反思、純吸引力語言、長期信任和承諾。

## trust（信任）

- 核心：安全感、依賴、承諾、誠實、未來導向、更深層的安全、關係成長。
- 好角度：什麼建立信任、什麼威脅安全、承諾意味著什麼、未來希望、勇敢的坦誠。
- 避免：調情能量、純粹策略性溝通、無共同安全感的一般自我發現。

## 難度遞進

- 第 1 級：輕鬆開場、低壓力、立即可回答、安全入點。
- 第 2 級：更多揭露、更具體、開始浮現需求、恐懼及有意義的記憶。
- 第 3 級：情感上更深入、更脆弱，但仍可在無壓力或表演的情況下回答。

## 親密指導

- 增加親近感、渴望、溫柔、身體意識或私密情感坦誠。
- 保持措辭為邀請性而非強制性。
- 避免露骨的性內容、強制、羞辱、嫉妒遊戲及愛的證明測試。
- 如不確定，選擇溫暖勝於大膽。
